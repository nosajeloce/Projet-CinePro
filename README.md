# CinePro Moub-Jas-You-Akr
## Nom du projet
Système de réservation de billets de cinéma (CinePro)

## Description
L'application Web développé permet aux utilisateurs d'effectuer des réservations de séances de visionnement de films au cinéma.
Au moment d'écrire ce README, l'application n'est pas fonctionnelle (aucune fonctionnalité qui marche).

## Configuration et installation
### Base de données
Étapes:
1. Exécuter le script sql/cinepro.sql
2. Exécuter le script PHP sql/insertion_donnees_comptes_client.php en ouvrant le fichier dans un environnement ayant un interpréteur PHP et qui stocke ce fichier.
3. Exécuter le script PHP sql/insertion_donnees_salles.php en ouvrant le fichier dans un environnement ayant un interpréteur PHP et qui stocke ce fichier.
4. Exécuter le script PHP sql/insertion_donnees_sieges.php en ouvrant le fichier dans un environnement ayant un interpréteur PHP et qui stocke ce fichier.
